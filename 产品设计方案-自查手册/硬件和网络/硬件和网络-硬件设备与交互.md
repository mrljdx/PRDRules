# 硬件和网络-硬件设备与交互

#### 硬件设备（主要针对移动端）

﻿

- **横 / 竖屏**----横竖屏切换时，页面布局及功能是否展示完整
- **适配情况**----关于不同分辨率机型的适配问题，是否备注清楚
- **SD卡（机身储存）**
  - 文件导入本地时，无SD卡、SD卡储存已满（无SD卡，机身储存已满）、储存位置等情况是否考虑并备注
  - 未获得SD卡（机身存储位置）权限时，导入本地时，提示内容

- **硬件 / 系统版本**----不同手机的物理按键、系统不同，是否需要备注说明
